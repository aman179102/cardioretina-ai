## Search Tips

### Finding Specific Information

**Use grep or search**:
```bash
# Search all docs for a term
grep -r "Grad-CAM" docs/

# Search for specific topic
grep -r "batch size" docs/
```

**Common search terms**:
- "training" - See training guides
- "evaluation" - See evaluation guides
- "architecture" - See technical docs
